## helper-tools

安装

```text
git clone https://github.com/alex-my/helper-tools.git
cd helper-tools
make mod
make i
```

## 命令列表

| 命令       | 缩写 | 作用                                             |
| ---------- | ---- | ------------------------------------------------ |
| PWD        | pwd  | 显示当前目录                                     |
| GitPullAll | gpa  | 遍历当前目录下的所有工程，分别执行 git pull      |
| SVNUpAll   | sua  | 遍历并更新当前目录下所有的 svn 项目，执行 svn up |
